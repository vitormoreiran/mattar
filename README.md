# Lab Mattar Prototype

This folder contains the first navigable prototype for the Lab Mattar presentation system.

## What's included

- `index.html`
- `styles.css`
- `content/pages.js`
- `app.js`
- `dla-acts.html`
- `content/dla-acts.js`
- `dla-acts.js`
- `mesa-stack.html`
- `content/mesa-acts.js`
- `content/mesa-vectors.js`
- `mesa-stack.js`

## Pilot pages

- `#/home`
- `#/commercial`
- `#/concept`
- `#/case`

## Notes

- This prototype is static so it can run without a Node.js toolchain.
- It validates the interface foundation, editorial block system, and integrated navigation.
- `dla-acts.html` is a standalone local study for DLA-based act visualization with local presets and palette mapping.
- `mesa-stack.html` is a mesa-specific study that stacks the local SVG levels and constrains particles by the real vector layers and fixed plate anchors.
